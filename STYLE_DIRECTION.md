# SocraChat GUI & Character Style Direction

## Core Style

Bold monochrome mobile GUI with large editorial typography, warm off-white surfaces, rounded content cards, simple black buttons, and a quiet black 3D cat mascot used only for emotional support moments.

The interface should feel functional, minimal, warm, witty, and brand-conscious. Cute but not childish. Playful but not decorative.

## GUI Rules

- Use a mostly monochrome system: black, white, warm off-white, light gray, dark gray.
- Use accent color rarely and intentionally.
- Typography should be bold, simple, and editorial.
- Large titles should carry the main message.
- Prefer short descriptions over long paragraphs.
- Create hierarchy with scale, weight, contrast, and spacing.
- Use mobile-first vertical layouts.
- Each screen should have one clear main message or one clear main action.
- Cards should be large rounded content blocks with generous padding.
- Primary buttons may be black filled buttons; secondary buttons should be quiet.

Avoid:
- colorful gradients
- saturated palettes
- pastel-heavy UI
- glossy effects
- glassmorphism
- strong shadows
- complex nested cards
- too many equally weighted blocks

## 3D Cat Mascot Rules

The 3D cat is not the main UI. It is a quiet brand assistant.

Use it only in:
- onboarding
- welcome screens
- empty states
- loading states
- completion screens
- error screens
- help or guide screens
- waiting states
- friendly tips
- result summary moments

Avoid it in:
- dense data screens
- complex forms
- payment/auth screens
- settings
- long lists
- comparison tables
- screens where the CTA must be the only focus

Character style:
- black 3D cat mascot
- soft clay or toy-like material
- rounded head and body
- simple white facial features
- cream or ivory headphones
- small cream crossbody bag
- minimal expression
- calm, gentle, quiet personality
- warm soft studio lighting
- no realistic fur
- no sharp details

Use only one mascot image per screen. The mascot supports the message; it must not overpower the product UI.
